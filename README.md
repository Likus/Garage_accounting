# Балконный учёт

***
##  Приветствую!
Вы находитесь на странице репозитория простейшего приложения для учёта товаров.

## Предыстория
Я решил начать торговлю на маркетплейсах. И захотелось мне вести учёт - что у меня есть, что у меня готово на продажу, и т.д. Я пытался вести записи в блокноте, но это было не совсем удобно. Разбираться в excel - таблицах мне не захотелось, а существующие решения требовали развёртывания сервера как минимум, чего я тоже не хотел. Так начало появляться это приложение.

## Возможности
- Формирование закупок
- Формирование товаров для продажи
- Учёт наличия купленного и готового к продаже
- Примерный подсчёт финансов (Расход, доход)

## Как пользоваться
- Открыть storage.html вашим любимым браузером
- Создать файл для базы данных в удобном для вас месте с помощью кнопки "Создать новую базу"
- Разобраться империческим методом (методом тыка)
-- Я делал приложение в первую очередь для себя, времени на туториал нету, извините(
- После окончания работы с учётом и возобновлении работы - открывать ранее созданный файл через кнопку "Открыть существующий файл"

Приложение создано с помощью нейросети Qwen
